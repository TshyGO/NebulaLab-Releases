# Nebula Graph Releases

This repository is the public distribution channel for **Nebula Graph** desktop builds.

## Download

Get the latest Windows build from the [Releases](https://github.com/TshyGO/NebulaGraph-Releases/releases/latest) page.

- Recommended: `Nebula Graph_x.y.z_x64-setup.exe`
- Alternative: `Nebula Graph_x.y.z_x64_en-US.msi`

## Asset Guide

- `setup.exe`: recommended Windows installer for most users
- `msi`: Windows Installer package
- `*.sig`: updater signatures used by the packaged desktop app
- `latest.json`: updater feed metadata for in-app updates

## About The Source Code Archives

GitHub automatically shows `Source code (zip)` and `Source code (tar.gz)` on every public Release.

Those archives are **not** the private Nebula Graph application source repository. They are only snapshots of this public distribution repo.

## Repository Role

- Private source/build repo: `TshyGO/NebulaGraph`
- Public release repo: `TshyGO/NebulaGraph-Releases`

This repo exists only to host installers, signatures, updater metadata, and release notes.
